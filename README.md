# Tita Media Portfolio

Se construye portafolio de una sola página full responsive y con accesibilidad.

## Herramientas usadas:

- HTML
- CSS
- JavaScript

## Adicionales
Se realizar fetch de un JSON, se colocan filtros por botones y por barra de búsqueda.
